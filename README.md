# 김비서 (kim1)

**말로 시키면 알아서 해주는 내 개인 비서.** 김비서는 두 곳에서 나를 돕는다.

- **폰(제미나이)** — 말을 걸면 바로 답하고, 바로 못 하는 일은 **"안 된다"고 하지 않고 할 일로 적어둔다.**
- **컴퓨터(자동 일꾼)** — 적어둔 할 일을 대신 처리하고, 그 결과와 진행 상태를 다시 알려준다.

내 지식(메모·사실·결정 같은 것)은 구글 드라이브에 차곡차곡 쌓이고, 할 일은 구글 캘린더에 쌓인다.
**앱을 새로 깔 필요 없다** — 이미 쓰는 구글 드라이브·캘린더·지메일과 제미나이만 있으면 된다.

> 이름 규칙: 비서마다 **비서코드**가 있다(이 비서는 `kim1`). 폰에서 "**비서 kim1**"이라고 부르면
> 그 비서의 지식과 할 일만 다룬다. 비서를 여러 명 두고 싶으면 프로젝트를 하나 더 만들면 된다.

## 김비서가 해주는 일

**바로 해주는 것**
- 📬 메일 **읽고 정리**해서 알려주기
- 📅 **일정 확인·추가·수정·삭제**
- 📁 드라이브 **문서 찾아 읽기 / 새로 만들기**
- 🧠 **내 지식 되찾기** — "그거 뭐였지?" 하면 저장해둔 내용을 찾아, 연결된 것까지 따라가며 답

**"할 일"로 적어두고 대신 처리하는 것** (바로는 못 하지만 거부하지 않음)
- ✉️ 메일 **보내기·답장**
- ✏️ 기존 **문서 고치기**
- 🧠 새 지식 **저장(정리해서 쌓기)**
- 🔎 **길게 조사**해서 결과 보고
- ⏰ **반복 일** — "평일 아침 9시에 해커뉴스 핫한 거 정리해서 메일 보내줘" 같은 것

## 할 일은 어떻게 흘러가나 (상태)

할 일 하나하나가 캘린더에 뜨고, 상태가 제목 앞에 보인다.

| 상태 | 뜻 |
|---|---|
| ⏳ 대기 | 아직 안 한 일 (예약 시간이 되면 시작) |
| 🔄 작업중 | 지금 하고 있음 |
| ✅ 성공 | 끝남 (결과는 그 일에 적힘) |
| ❌ 실패 | 못 함 (이유가 적힘) |
| ❓ 정보필요 | 뭔가 더 알려줘야 함 (뭐가 필요한지 적혀 있음) |

**❓ 정보필요**가 뜨면, 내가 말로 알려주기만 하면 다시 **⏳ 대기**로 돌아가 알아서 이어서 한다.
모든 진행 내용은 그 할 일에 **기록으로 남는다.**

## 어떻게 말 걸까 (맛보기)

- "비서 kim1, 지금 비서코드 뭐야?" → "kim1 입니다."
- "그때 정한 거 뭐였지?" → 저장해둔 지식에서 찾아 답
- "이 메일 김대리한테 답장해줘" → "큐에 넣었어요" (컴퓨터 일꾼이 처리)
- "지금 진행중인 일 뭐야?" / "정보필요로 멈춘 일 뭐 있어?" → 상태 요약
- "이건 바로 말고 길게 조사해줘" → 할 일로 적어두고, 조사 끝나면 그 일에 결과 보고

더 많은 예시는 [docs/examples.md](docs/examples.md) 참고.
